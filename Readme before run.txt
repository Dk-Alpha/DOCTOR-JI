Steps to run the project:

1. open the DoctorJI.ipynb file for running the whole project

2.Dont interfere with any of the repo files

3.drugs_for_common_treatment.csv file is used for training the model

4. The project uses ThirdAI API for training models. (Appropriate license key is required for it)

5. Dont run the "model.cold_start(.............)" cell it has already been trained

6. directly load and run the next cell for loading the existing trained model
